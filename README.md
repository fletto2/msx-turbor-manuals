# MSX turboR Manual — English translation

English translation of the **MSX turboR Technical Hand Book**, translated from the
Japanese original and cross-checked against the relevant component datasheets.

The manual is provided as **PDF** and **ODT**, with the source **Markdown** under
[`markdown/`](markdown). The [`datasheets/`](datasheets) folder holds datasheets for the
turboR's main ICs — YM2149 (PSG), YM2413 (OPLL / MSX-MUSIC) and V9958 (MSX-VIDEO VDP).

This is a datasheet-backed translation, provided for preservation and reference. All
trademarks belong to their respective owners.
